# el3iraky padball

Mobile-first single-page application for the Egyptian Padbol community.

## Tech stack

- React + Vite
- Tailwind CSS
- Framer Motion
- Lucide icons

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Screens

- **Welcome / Auth Gate** – Login or create an account
- **Home** – Tournament list and match schedule
- **Profile** – View and edit player data and stats
- **Scoreboard** – Top players leaderboard
- **Friends** – Searchable friends list with active toggles
